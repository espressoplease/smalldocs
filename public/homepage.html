Install

One command. No npm, no root.

npm’s global folder is often owned by root, so a regular install crashes with a permission error; this script writes into a folder you own under your home directory instead.

$ curl -fsSL https://smalldocs.org/install | sh

Needs Node.js. Runs on macOS and Linux. Prefer npm? npm i -g sdocs-dev still works as a fallback.

If sdoc isn’t found after installing, add it to your PATH
1

Check which shell you’re using

Run this in your terminal:

echo $SHELL # /bin/zsh or /bin/bash or /usr/bin/fish
2

Open your shell’s config file

  • bash · ~/.bashrc
  • zsh · ~/.zshrc
  • fish · ~/.config/fish/config.fish
3

Add the SmallDocs directory to PATH

Add this line to that file (fish users see below):

export PATH="$HOME/.sdocs/bin:$PATH"

For fish, use:

fish_add_path $HOME/.sdocs/bin
4

Reload your shell

Either open a new terminal window, or run:

source ~/.zshrc # or ~/.bashrc
