Install SmallDocs.

SmallDocs is open source and free to use. It installs as a small command-line tool under your home directory.

$ curl -fsSL https://smalldocs.org/install | sh

The script runs on macOS and Linux. On Windows, or via npm, install with npm i -g sdocs-dev.

Why a custom install script

npm’s global folder is usually owned by root, so a regular install can fail with a permission error. This script installs SmallDocs into ~/.sdocs, a folder you own, so it doesn’t need sudo.

If sdoc isn’t found, add it to your PATH

Check which shell you’re using

Run this in your terminal:

echo $SHELL # /bin/zsh or /bin/bash or /usr/bin/fish

Open your shell’s config file

  • bash · ~/.bashrc
  • zsh · ~/.zshrc
  • fish · ~/.config/fish/config.fish

Add the SmallDocs directory to PATH

Add this line to that file (fish users see below):

export PATH="$HOME/.sdocs/bin:$PATH"

For fish, use:

fish_add_path $HOME/.sdocs/bin

Reload your shell

Either open a new terminal window, or run:

source ~/.zshrc # or ~/.bashrc
